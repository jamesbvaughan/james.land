How I make this site

2024-10-14

This is the most simple site that I operate.

It's hosted here on GitHub and is served by GitHub Pages.

There is no build step. There is no static site generator. There are no templates.

There is some HTML, a bit of CSS, and some occasional JavaScript.

If I want to change the header or footer, I either update each page's HTML or I accept the inconsistency.
